# Module RUNTIME

## Rôle
Génère automatiquement le code Django et déploie.

## Processus automatique

1. Utilisateur termine dans Studio
2. Clique "Déployer"
3. Runtime génère:
   - models.py
   - serializers.py
   - views.py
   - urls.py
   - admin.py
4. Déploie avec Docker

## Utilisateur interagit ?
❌ NON - Tout automatique
